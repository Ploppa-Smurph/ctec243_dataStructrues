10.01 - Graph Interface

    Graph - nodes that connect to other nodes, has edges and vertices
    Vertex - node of a graph
    Edge (arc) - Pair of vertices that represents a connection between 2 vertices in the graph
    Undirected Graph - edges have NO direction
    Directed Graph (Digraph) - graph in which edge is directed from 1 vertex to another

    Adjacent Vertices - 2 vertices (nodes) that are connected by an edge 
    Path - the sequence of vertices that connects 2 vertices in a graph (route from 1 node to another non-adjacent node)
    Complete Graph - each vertex is connected to each other vertex
    Weighted Graph - graph where each 'edge' carries a value, ex: the map with milage on edge between nodes


10.02 - Implementations of Graphs

    Array-Based implementation - int for # of vertices, array for the vertices, and 2-D array for edges

    Linked Implementation - each node points to link list

    -- Depth-First = use stack, goes deeper before wider
    -- Breadth First (level-by-level) = use queue, goes wide before deep
